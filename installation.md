# Installation

Either from the console:

~~~ shell
    $ composer require ptlis/conneg-bundle:"~3.0.0"
~~~

Or by Editing composer.json:

~~~json
    {
        "require": {
            ...
            "ptlis/conneg-bundle": "~3.0.0",
            ...
        }
    }
~~~

Followed by a composer update:

~~~ shell
    $ composer update
~~~
